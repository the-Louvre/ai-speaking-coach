最值得改的一句

Original: I improved the route flow.

Better: I optimized the route planning experience and reduced route search time by 30%.

原因:“route planning experience” 更自然,“30%” 让你的成果可衡量。

下一轮只练一件事

不是说更多,而是说准一个数字。

推荐复练句
  • I optimized the route planning experience and reduced route search time by 30%.
  • This project improved user navigation efficiency.
  • My main contribution was designing the route interaction flow.

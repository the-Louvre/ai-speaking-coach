# 开发与交付计划

## 合规窗口

- 开始时间：2026-06-05 00:00，Asia/Shanghai (UTC+08:00)
- 截止时间：2026-06-08 23:59，Asia/Shanghai (UTC+08:00)
- 规则：所有 commit 时间戳必须落在该窗口内。

## 交付物

- GitHub/Gitee 可公开访问仓库，当前采用 GitHub 私有仓库开发，截止后公开。
- 持续 PR 与 commit 记录。
- README 文档，包含依赖、运行方式、原创说明和 demo 信息。
- demo 视频，展示场景选择、语音对话、纠错、发音评测和课后总结。

## PR 拆分

1. 仓库初始化与合规文档。
2. 项目脚手架与基础页面壳。
3. 场景选择与练习流程。
4. 语音录制与实时对话。
5. 发音评测与纠错反馈。
6. 课后总结与量化反馈。
7. demo 视频、README 复现说明与最终验收。

## 每次 PR 检查项

- PR 只包含单一功能。
- PR 描述包含功能描述、实现思路、测试方式、依赖与来源说明。
- 主分支合并后保持可运行状态。
- 新增依赖已在 README 或 PR 描述中说明。
- 若复用本人历史代码或公开示例，必须写明来源和改动。
